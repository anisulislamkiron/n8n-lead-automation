# 🤖 n8n Lead Automation: Google Sheets → Gmail

Automated lead follow-up system that instantly sends personalized emails when new rows are added to Google Sheets.

## 📋 Problem Solved
- Manual lead follow-up caused 2-6 hour response delays
- Inconsistent email messaging
- Time wasted on repetitive tasks

## ✅ Solution
Built a zero-code n8n workflow that:
- Triggers on new Google Sheet rows (every 60 seconds)
- Maps contact data dynamically
- Sends personalized Gmail auto-replies
- **Result:** Response time reduced from hours to <60 seconds
